# Shop_Sales_Margin_Analysis_Dashboard
This project is a sports shop sales dashboard built using SQL and Power BI. The goal was to identify margin and sale amount of  key products and analyze pricing.
# Business Problem
The objective of this analysis was evaluate sports shop sales performance and identify the products pricing and profitability to support better pricing decisions. 
# Dataset
The dataset is AdventureWorks database (SQL Server sample database).
# Tools Used
•	SQL 
•	Power BI (Power Query, DAX)
•	Charts
# Analysis Process
•	Data preparation in SQL – selected tables and built analytical dataset using joins. Removed irrelevant attributes and retained only variables relevant to revenue, cost.
•	Data preparation in Power BI – merged tables using Power Query, built a product hierarchy and prepared a clean analytical model. Removed redundant attributes and created key KPIs (Margin %) using DAX .
•	Data Visualization in Power BI – used Cards, tables, line and column charts to visualize data. Identified top and bottom performing products
# Key Insights
•	Bike Mountain-Black 40  generated the highest sales amount
•	Racing Socks have the highest % Margin
•	Bike Mountain-Black 38 has the highest margin amount
•	Patch Kit/8 is the most frequently purchased product
•	Bikes category generated is the most popular
•	In October 2025 recorded the highest sales amount
•	Sales increase in 2025
# Recommendations
•	Focus marketing efforts on high-performing products such as bikes
•	Review pricing strategy for Road-250 Red bikes to improve margin performance.
•	Analyze seasonality patterns to leverage peak periods like October
